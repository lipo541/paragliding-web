# VS Code рЃљрЃњрЃћрЃюрЃбрЃћрЃЉрЃў - рЃАрЃарЃБрЃџрЃў рЃњрЃќрЃљрЃЏрЃЎрЃЋрЃџрЃћрЃЋрЃў

## рЃарЃљ рЃљрЃарЃўрЃА VS Code рЃљрЃњрЃћрЃюрЃбрЃћрЃЉрЃў?

VS Code-рЃўрЃА рЃљрЃ«рЃљрЃџрЃў рЃљрЃњрЃћрЃюрЃбрЃћрЃЉрЃў рЃљрЃарЃўрЃА рЃ«рЃћрЃџрЃЮрЃЋрЃюрЃБрЃарЃў рЃўрЃюрЃбрЃћрЃџрЃћрЃЦрЃбрЃўрЃЌ рЃњрЃљрЃФрЃџрЃўрЃћрЃарЃћрЃЉрЃБрЃџрЃў рЃљрЃАрЃўрЃАрЃбрЃћрЃюрЃбрЃћрЃЉрЃў, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃЊрЃљрЃњрЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃўрЃљрЃю рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃљрЃерЃў рЃЊрЃљ рЃњрЃљрЃюрЃЋрЃўрЃЌрЃљрЃарЃћрЃЉрЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃерЃў. рЃўрЃАрЃўрЃюрЃў рЃгрЃљрЃарЃЏрЃЮрЃљрЃЊрЃњрЃћрЃюрЃћрЃю GitHub Copilot-рЃўрЃА рЃљрЃ«рЃљрЃџ рЃЌрЃљрЃЮрЃЉрЃљрЃА рЃЊрЃљ рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃћрЃю рЃБрЃцрЃарЃЮ рЃўрЃюрЃбрЃћрЃџрЃћрЃЦрЃбрЃБрЃљрЃџрЃБрЃа рЃЊрЃљ рЃЎрЃЮрЃюрЃбрЃћрЃЦрЃАрЃбрЃќрЃћ рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџ рЃЊрЃљрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљрЃА.

## рЃљрЃњрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃбрЃўрЃърЃћрЃЉрЃў

### 1. **@workspace** рЃљрЃњрЃћрЃюрЃбрЃў
- **рЃЊрЃљрЃюрЃўрЃерЃюрЃБрЃџрЃћрЃЉрЃљ**: рЃЏрЃЌрЃџрЃўрЃљрЃюрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЎрЃЮрЃюрЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃњрЃљрЃњрЃћрЃЉрЃљ
- **рЃарЃЮрЃЊрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃЮрЃЌ**: 
  - рЃарЃЮрЃфрЃљ рЃњрЃАрЃБрЃарЃЌ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ рЃЏрЃЌрЃћрЃџ рЃърЃарЃЮрЃћрЃЦрЃбрЃќрЃћ
  - рЃарЃЮрЃфрЃљ рЃћрЃФрЃћрЃЉрЃЌ рЃцрЃљрЃўрЃџрЃћрЃЉрЃА рЃљрЃю рЃЎрЃЮрЃЊрЃўрЃА рЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃА
  - рЃарЃЮрЃфрЃљ рЃњрЃАрЃБрЃарЃЌ рЃарЃћрЃцрЃљрЃЦрЃбрЃЮрЃарЃўрЃюрЃњрЃў рЃЏрЃЌрЃћрЃџ рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў

**рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў**:
```
@workspace рЃАрЃљрЃЊ рЃљрЃарЃўрЃА рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃБрЃџрЃў рЃЉрЃБрЃЦрЃўрЃюрЃњрЃўрЃА рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃў?
@workspace рЃарЃЮрЃњрЃЮрЃа рЃЋрЃљрЃЎрЃћрЃЌрЃћрЃЉ SEO рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃљрЃфрЃўрЃљрЃА рЃљрЃЏ рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў?
```

### 2. **@vscode** рЃљрЃњрЃћрЃюрЃбрЃў
- **рЃЊрЃљрЃюрЃўрЃерЃюрЃБрЃџрЃћрЃЉрЃљ**: VS Code рЃарЃћрЃЊрЃљрЃЦрЃбрЃЮрЃарЃўрЃА рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃЌрЃљрЃю рЃЊрЃљ рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃЌрЃљрЃю рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљ
- **рЃарЃЮрЃЊрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃЮрЃЌ**:
  - рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃўрЃА рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ
  - рЃерЃЮрЃарЃЌрЃЦрЃљрЃЌрЃћрЃЉрЃўрЃА рЃњрЃљрЃњрЃћрЃЉрЃљ
  - рЃњрЃљрЃцрЃљрЃарЃЌрЃЮрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ

**рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў**:
```
@vscode рЃарЃЮрЃњрЃЮрЃа рЃЊрЃљрЃЋрЃљрЃДрЃћрЃюрЃЮ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃарЃћрЃЉрЃљ рЃерЃћрЃюрЃљрЃ«рЃЋрЃўрЃАрЃљрЃА?
@vscode рЃарЃЮрЃЏрЃћрЃџрЃў рЃњрЃљрЃцрЃљрЃарЃЌрЃЮрЃћрЃЉрЃћрЃЉрЃў рЃЏрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ React-рЃўрЃАрЃЌрЃЋрЃўрЃА?
```

### 3. **@terminal** рЃљрЃњрЃћрЃюрЃбрЃў
- **рЃЊрЃљрЃюрЃўрЃерЃюрЃБрЃџрЃћрЃЉрЃљ**: рЃбрЃћрЃарЃЏрЃўрЃюрЃљрЃџрЃерЃў рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃљ
- **рЃарЃЮрЃЊрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃЮрЃЌ**:
  - рЃЎрЃЮрЃЊрЃўрЃА рЃљрЃгрЃДрЃЮрЃЉрЃљ рЃЊрЃљ рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ
  - Git рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃў
  - рЃърЃљрЃЎрЃћрЃбрЃћрЃЉрЃўрЃА рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ

**рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў**:
```
@terminal рЃњрЃљрЃБрЃерЃЋрЃў рЃбрЃћрЃАрЃбрЃћрЃЉрЃў рЃЊрЃљ рЃљрЃЕрЃЋрЃћрЃюрЃћ рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃў
@terminal рЃЊрЃљрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћ axios рЃЊрЃљ рЃњрЃљрЃюрЃљрЃљрЃ«рЃџрЃћ package.json
```

## рЃљрЃњрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃБрЃърЃўрЃарЃљрЃбрЃћрЃАрЃЮрЃЉрЃћрЃЉрЃў рЃЕрЃЋрЃћрЃюрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў

### 1. **рЃАрЃгрЃарЃљрЃцрЃў рЃюрЃљрЃЋрЃўрЃњрЃљрЃфрЃўрЃљ рЃЎрЃЮрЃЊрЃерЃў**
Next.js рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃўрЃДрЃЮрЃА рЃарЃЌрЃБрЃџрЃў рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃЌ. рЃљрЃњрЃћрЃюрЃбрЃћрЃЉрЃў рЃЊрЃљрЃњрЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃўрЃљрЃю:
- рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃАрЃгрЃарЃљрЃцрЃљрЃЊ рЃърЃЮрЃЋрЃюрЃљрЃерЃў
- API route-рЃћрЃЉрЃўрЃА рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃерЃў
- Supabase рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃўрЃА рЃњрЃљрЃњрЃћрЃЉрЃљрЃерЃў

### 2. **рЃЎрЃЮрЃЊрЃўрЃА рЃ«рЃљрЃарЃўрЃАрЃ«рЃўрЃА рЃњрЃљрЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃћрЃЉрЃљ**
- рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃарЃћрЃцрЃљрЃЦрЃбрЃЮрЃарЃўрЃюрЃњрЃўрЃА рЃерЃћрЃЌрЃљрЃЋрЃљрЃќрЃћрЃЉрЃћрЃЉрЃў
- рЃЎрЃЮрЃЊрЃўрЃА рЃАрЃбрЃўрЃџрЃўрЃА рЃЎрЃЮрЃюрЃАрЃўрЃАрЃбрЃћрЃюрЃбрЃЮрЃЉрЃљ
- рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃћрЃЉрЃўрЃА рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃЋрЃџрЃћрЃюрЃљ

### 3. **рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃўрЃА рЃњрЃћрЃюрЃћрЃарЃљрЃфрЃўрЃљ**
- JSDoc рЃЎрЃЮрЃЏрЃћрЃюрЃбрЃљрЃарЃћрЃЉрЃўрЃА рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃерЃћрЃЦрЃЏрЃюрЃљ
- README рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃњрЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃљ
- TypeScript рЃбрЃўрЃърЃћрЃЉрЃўрЃА рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃўрЃарЃћрЃЉрЃљ

## рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃБрЃџрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў рЃЕрЃЋрЃћрЃюрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃАрЃЌрЃЋрЃўрЃА

### рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў 1: рЃЉрЃБрЃЦрЃўрЃюрЃњ рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃАрЃЌрЃљрЃю рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљ
```
рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў: @workspace рЃарЃЮрЃњрЃЮрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃЉрЃБрЃЦрЃўрЃюрЃњрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ?
рЃљрЃњрЃћрЃюрЃбрЃў: рЃЉрЃБрЃЦрЃўрЃюрЃњрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ рЃњрЃљрЃюрЃџрЃљрЃњрЃћрЃЉрЃБрЃџрЃўрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњрЃюрЃљрЃўрЃарЃљрЃЊ:
- /app/booking/page.tsx - рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃЉрЃБрЃЦрЃўрЃюрЃњрЃўрЃА рЃњрЃЋрЃћрЃарЃЊрЃў
- /components/booking/ - рЃЉрЃБрЃЦрЃўрЃюрЃњрЃўрЃА рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃћрЃЉрЃў
- /lib/supabase/bookings.ts - Supabase рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃљ
```

### рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў 2: SEO рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃљрЃфрЃўрЃљ
```
рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў: @workspace рЃарЃЮрЃњрЃЮрЃа рЃЊрЃљрЃЋрЃљрЃЏрЃљрЃбрЃЮ meta tags рЃљрЃ«рЃљрЃџ рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ?
рЃљрЃњрЃћрЃюрЃбрЃў: рЃљрЃЏ рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў meta tags рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњрЃюрЃљрЃўрЃарЃљрЃЊ:
- рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ Next.js Metadata API
- рЃюрЃљрЃ«рЃћрЃЌ рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў app/[locale]/layout.tsx рЃцрЃљрЃўрЃџрЃерЃў
- рЃЦрЃљрЃарЃЌрЃБрЃџрЃў рЃЊрЃљ рЃўрЃюрЃњрЃџрЃўрЃАрЃБрЃарЃў рЃЋрЃћрЃарЃАрЃўрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃБрЃџрЃў meta tags
```

## рЃарЃЮрЃњрЃЮрЃа рЃЋрЃўрЃгрЃДрЃЮрЃЌ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљ рЃљрЃњрЃћрЃюрЃбрЃћрЃЉрЃЌрЃљрЃю

- рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ Chat рЃърЃљрЃюрЃћрЃџрЃў (`Ctrl+Alt+I` рЃљрЃю `Cmd+Alt+I`)

**рЃфрЃБрЃЊрЃў рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў**:
```
рЃарЃЮрЃњрЃЮрЃа рЃЋрЃљрЃЎрЃћрЃЌрЃЮ рЃЎрЃЮрЃЊрЃўрЃарЃћрЃЉрЃљ? (рЃФрЃљрЃџрЃўрЃљрЃю рЃќрЃЮрЃњрЃљрЃЊрЃў)
```

### 3. **рЃЏрЃљрЃЦрЃАрЃўрЃЏрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ**

- **рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃЎрЃЮрЃюрЃбрЃћрЃЦрЃАрЃбрЃў**: рЃЏрЃўрЃБрЃЌрЃўрЃЌрЃћрЃЌ рЃарЃљ рЃбрЃўрЃърЃўрЃА рЃЊрЃљрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљ рЃњрЃГрЃўрЃарЃЊрЃћрЃЉрЃљрЃЌ

## рЃљрЃњрЃћрЃюрЃбрЃћрЃЉрЃЌрЃљрЃю рЃЏрЃБрЃерЃљрЃЮрЃЉрЃўрЃА рЃАрЃљрЃБрЃЎрЃћрЃЌрЃћрЃАрЃЮ рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃћрЃЉрЃў

## рЃерЃћрЃќрЃдрЃБрЃЊрЃЋрЃћрЃЉрЃў рЃЊрЃљ рЃарЃЕрЃћрЃЋрЃћрЃЉрЃў

### рЃерЃћрЃќрЃдрЃБрЃЊрЃЋрЃћрЃЉрЃў:
- рЃљрЃњрЃћрЃюрЃбрЃћрЃЉрЃў рЃљрЃа рЃљрЃарЃўрЃљрЃю 100% рЃќрЃБрЃАрЃбрЃў

- рЃЊрЃўрЃЊ рЃцрЃљрЃўрЃџрЃћрЃЉрЃЌрЃљрЃю рЃЏрЃБрЃерЃљрЃЮрЃЉрЃљ рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃўрЃДрЃЮрЃА рЃюрЃћрЃџрЃў

### рЃарЃЕрЃћрЃЋрЃћрЃЉрЃў:

- РюЁ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ рЃљрЃњрЃћрЃюрЃбрЃћрЃЉрЃў рЃАрЃгрЃљрЃЋрЃџрЃўрЃАрЃЌрЃЋрЃўрЃА рЃЊрЃљ рЃњрЃљрЃњрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА
- РюЁ рЃЊрЃљрЃБрЃЌрЃЏрЃћрЃЌ рЃЊрЃарЃЮ рЃЎрЃљрЃарЃњрЃљрЃЊ рЃцрЃЮрЃарЃЏрЃБрЃџрЃўрЃарЃћрЃЉрЃБрЃџ рЃЎрЃўрЃЌрЃ«рЃЋрЃћрЃЉрЃА

- РЮї рЃљрЃа рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮрЃЌ рЃАрЃћрЃюрЃАрЃўрЃбрЃўрЃБрЃарЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА
