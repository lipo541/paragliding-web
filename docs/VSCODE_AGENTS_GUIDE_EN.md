# VS Code Agents - Complete Guide

## What are VS Code Agents?

VS Code agents are AI-powered assistants that help you with coding and development processes. They represent the new generation of GitHub Copilot and provide more intelligent and context-aware assistance.

## Types of Agents

### 1. **@workspace** Agent
- **Purpose**: Understanding the entire project context
- **When to use**: 
  - When you need information about the whole project
  - When searching for files or code sections
  - When performing refactoring across the project

**Examples**:
```
@workspace where is the booking component defined?
@workspace how do I perform SEO optimization in this project?
```

### 2. **@vscode** Agent
- **Purpose**: Working with VS Code editor settings and features
- **When to use**:
  - Configuration settings
  - Understanding shortcuts
  - Installing extensions

**Examples**:
```
@vscode how do I set up automatic formatting on save?
@vscode which extensions do I need for React?
```

### 3. **@terminal** Agent
- **Purpose**: Executing commands in the terminal
- **When to use**:
  - Building and testing code
  - Git operations
  - Installing packages

**Examples**:
```
@terminal run tests and show results
@terminal install axios and update package.json
```

## How to Start Working with Agents

### 1. **Activation**
- Ensure you have GitHub Copilot Chat extension installed
- Open Chat panel (`Ctrl+Alt+I` or `Cmd+Alt+I`)

### 2. **Asking Effective Questions**
**Good examples**:
```
@workspace where is user authentication defined?
@workspace how do I add a new location to the locations list?
@workspace what is the glass morphism style convention in this project?
```

**Bad examples**:
```
How do I code? (too general)
Write me a program (no context)
```

### 3. **Maximizing Usage**
- **Be specific**: Mention file names or components
- **Use context**: Specify what type of help you need
- **Iterate**: If the answer doesn't help, rephrase your question
